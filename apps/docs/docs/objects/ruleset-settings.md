# RulesetSettings

```json
{
  "foodSpawnChance": 25,
}
```

Детальные настройки параметров геймплея.

| **Свойство**                    | **Тип** | **Описание**                                                                                                                      |
| ------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **foodSpawnChance**             | integer  | Вероятность спавна новой еды.                                           |
  
Также, объект может содержать иные свойства, неактуальные в предстоящем турнире.
